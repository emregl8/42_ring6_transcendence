=========== İLK KURULUM ===========

1. aşağıdaki paketleri indirin
sudo apt install -y curl wget git openssl make npm

2. docker engine'i kurun
https://docs.docker.com/engine/install

3. k3s'i kurun
curl -sfL https://get.k3s.io | sh -
mkdir -p ~/.kube
sudo cp /etc/rancher/k3s/k3s.yaml ~/.kube/config
sudo chown $(id -u):$(id -g) ~/.kube/config
echo 'export KUBECONFIG=$HOME/.kube/config' >> ~/.bashrc
source ~/.bashrc

4. kullanıcınızı gerekli gruplara ekleyin
sudo usermod -aG docker $USER
üstteki komutu yazdıktan sonra bilgisayarı yeniden başlatmanız gerek

5. "make dev" ile sunucuyu başlatın
https://transcendence.local


=========== EKİBE NOTLAR =========== (buraya not bırakabilirsiniz)

1. "make clean" namespace'i sildiği için volume'ler de silinir ve tüm data(veritabanı) kaybolur.
Data'nın korunmasını istiyorsanız pod'ları "make rebuild-frontend" veya "make rebuild-postgres" gibi rebuild komutlarını kullanın.

2. Proje her çalıştırıldığında parolalar rastgele oluşturulup .env'e kaydediliyor.

3. Proje debian sanal makinede denendi. Farklı dağıtımlarda sorun çıkabilir.

========== GÜVENLİK/PRODUCTION NOTLARI =========

1. unseal key'ler development için secrets olarak eklendi ve root token silindi. production'da kms/hsm kullanılacak.
2. memory lock ayarlanacak
3. imagePullPolicy değiştirilecek
4. letsencrypt ayarlanacak
5. DTO sınıfları yazılacak
6. authentication:anonymous false yapılacak
7. kube-apiserver -> kubelet iletişimi incelenecek
8. admission controller eklenecek
